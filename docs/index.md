# The Blimp Capstone

Technical documentation for the **Blimp** 
(Innopolis Robotics Society).

## What is this project?

> An airship is a lighter-than-air aircraft. It's designed for indoor flight and operates in GPS-denied areas using Nooploop's ultra-wideband localization technology.
